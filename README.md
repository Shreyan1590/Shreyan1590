## 📂 Highlighted Production Deployments

### 🌐 [Thendral Kulirbanagam](https://github.com/Shreyan1590) — Commercial Brand Portal
*   **Objective**: Build a high-performance web system for a retail beverage storefront.
*   **Tech Stack**: Next.js, TailwindCSS, Framer Motion, and Firebase.
*   **Outcome**: Implemented responsive product showcases alongside premium, native custom animation systems resulting in zero layout shift (CLS) and lightning-fast edge loading speeds.

### 🛡️ [Data Leakage Detection Engine](https://github.com/Shreyan1590) — Security Automation
*   **Objective**: Develop a pipeline to automatically inspect, locate, and restrict sensitive data leaks.
*   **Tech Stack**: Python, RegEx Engines, File Stream Parsers.
*   **Outcome**: Engineered highly customizable scripts capable of scanning thousands of complex configuration and codebase files for exposed API keys, private tokens, and credentials in seconds.

### 👥 [Tech Slashers Platform](https://github.com/Shreyan1590) — Hackathon Central Hub
*   **Objective**: Designed the digital branding and interactive team application repository for highly competitive hackathon pipelines.
*   **Tech Stack**: React, TypeScript, and interactive custom canvas-driven elements.
*   **Outcome**: Allowed instant onboarding of team tasks and project tracking while delivering a gorgeous "cyberpunk terminal" aesthetic.
